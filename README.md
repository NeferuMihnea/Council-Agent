# CONSILIUL – Multi-Agent AI Proposal System

A multi-agent AI system built on Microsoft Copilot Studio that 
automates end-to-end project proposal generation.

## Built with
- Microsoft Copilot Studio
- Power Automate
- Microsoft Dataverse
- Microsoft 365

## How to import
1. Download Consiliul_System.zip
2. Go to make.powerapps.com → Solutions → Import
3. Upload the .zip and follow the steps

## Agents
- Orchestrator (Consiliul)
- MS-CandidatesAgent
- MS-CandidatesRecommendation Agent
- MS-Finance Agent
- MS-PastProjectsHistory
- MS-MarketingProposal Agent
