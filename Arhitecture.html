Independent Flow
🆕 HR Agent
  • Receives new employee CV
  • Extracts: name, role, skills, certs, experience
  • Creates record in Dataverse
  • Immediately available to Resource Mgmt Agent
⬇ Writes to Dataverse via PA Flow
⬇⬇⬇
Specialist Agents — Invoked in parallel by Orchestrator
📁 Past Projects Agent
  • Searches past project portfolio
  • Identifies similar engagements by industry, technologies, scope
  • Extracts: team composition, methodology, outcomes, lessons learned
  • Returns relevant experience to Orchestrator for proposal narrative
📂 Knowledge: past proposal documents (Word/PDF)
👥 Resource Management Agent
  • Receives required skills list from RFP
  • Calls PA flow → queries Dataverse for matching employees
  • Filters by skill proficiency & availability
  • Returns: proposed team with justification per role
⚡ Action: PA Flow → Dataverse (employees + skills + availability)
💶 Finance Agent
  • Receives proposed team from Resource Mgmt Agent
  • Calls PA flow → gets salary / daily rate per employee from Dataverse
  • Calculates total project cost (days × rate × allocation)
  • Compares cost vs. client budget from RFP
  • Returns: profitable ✅ / not profitable ❌ + breakdown
⚡ Action: PA Flow → Dataverse (salary + daily rates)
